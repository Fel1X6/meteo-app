Weather app ir lietotne, kurā lietotājs ievada pilsētu, izmantojot publisku API, var skatīt laika prognozes šodienai vai nākamajai dienai.

**Izmantots:**
Open Meteo API - weather data   
  Izveidots ar HTML, JavaScript, CSS ReactJS, ViteJS un Fetch API

**Tīmekļa lietojumprogrammas funkcionalitāte:**  
Meklēt laika prognozes pēc pilsētas nosaukuma,  
Attēlot laika prognozes šodienai un nākamajai dienai,  
Stundas prognoze (pirmās 24 stundas), kopuma (48 stundas),  
Attēlošanas režīmi: (Kartes skats, Tabulas skats)

**Kļūdu apstrāde:**  
1) Ja lietotājs ievadīja tukšu ievades lauku
2) Ja lietotājs ievadīja nepareizu pilsētas nosaukumu
3) Ja lietotāja pusē ir API nepieejamības kļūdas vai tīkla kļūdas
